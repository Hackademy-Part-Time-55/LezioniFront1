Una funzione è un blocco di codice "riutilizzabile" a cui è possibile assegnare un nome


Possimo scomporre un programma complesso in piccole parti più semplici.
Queste piccole parti potrebbe essere le funzione (programmazione procedurale)


Possiamo immagina una funzione come un mini-programma che può richiedere dati in input e ritornare un dato in output



// Dichiarazione della funzione
function nomeDellaFunzione(parametro1, parametro2, parametro3) {

}


// Invocare (chiamare) la funzione
nomeDellaFunzione(argomento1, argomento2, argomento3);



Il nomeDellaFunzione deve ben descrivere "cosa" la funzione fa (l'azione) 



return fa 2 cose:
- termina l'esecuzione della funzione
- fa fuoriuscire un VALORE dal blocco di codice della funzione










lo scope di una variabile (ambito di visibiltà) specifica in quale righe di codice la variabile è accessibile (visibile)
 
una variabile dichiara con la keyword var ha scope a livello di funzione entro cui è stata dichiarata
una variabile dichiara con la keyword let (così come una costante dichiarata con const) ha scope a livello di blocco di codice entro cui è stata dichiarata
