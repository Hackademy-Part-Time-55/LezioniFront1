attributi globali:
    id  un modo per identificare in maniera univoca un certo elemento
    class un insieme di elementi
    style



tag semantici cioè contenenitori block aventi "ruolo" ben specifico:
    nav
    header presenta del contenuto introduttivo relativo al suo elemento parent
    main
    article
    section
    footer presenta del contenuto secondario relativo al suo elemento parent
    aside presenta del contenuto indirettamente correlato al contenuto del main (usato come sidebar oppure pubblicità)

Un article è un contenitore per del contenuto informativo (per l'utente):
- indipendete dal contenuto di altri article
- self-contained cioè l'informazione è tutta contenuta al suo interno
Ogni article deve avere un titolo (un heading tag)


section è un generico contenitore utilizzato per raggruppare delle informazioni legate tra di loro a livello tematico (categorie)
Ogni section deve avere un titolo (un heading tag)



form è un contenitore di tipo block

Gli elementi input sono inline e self closing tag.
Sono presenti input di type differente:
- text (il più generico)
- number
- email
- password

- checkbox

- file
